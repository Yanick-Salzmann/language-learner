## Requirements

- Java 17 or higher
- Maven 3.6 or higher

## Running the Application

### Using Maven

```bash
# Compile the application
mvn clean compile

# Run tests
mvn test

# Start the application
mvn spring-boot:run
```

The application will start on `http://localhost:8080`

### Using Java

```bash
# Build the JAR
mvn clean package

# Run the JAR
java -jar target/ai-language-learner-1.0-SNAPSHOT.jar
```
